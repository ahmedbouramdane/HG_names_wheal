# Classroom Names Wheal

This is a classroom names wheal application for 1st-year baccalaureate students in Morocco.

## Features

- Spin a wheal to select a random name from a class
- Save and restore class data
- Reset class data to defaults
- Factory reset all data

## Tech Stack

- HTML
- CSS
- JavaScript

## Usage

1. Open `index.html` in your browser
2. Select a class from the dropdown
3. Spin the wheal to select a random name
4. Reset data if needed

## Files

- `index.html`: Main application file
- `app.js`: Application logic
- `wheel.js`: Wheal rotation logic
- `storage.js`: Data storage and retrieval
- `style.css`: Application styling
- `swal-custom.js`: Custom SweetAlert2 styles
- `swal.min.js`: SweetAlert2 library
